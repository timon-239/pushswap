## Description

Push Swap is a sorting algorithm project from the 42 curriculum. The goal is to sort a stack of integers in ascending order using a limited set of stack operations while minimizing the total number of moves.

The project focuses on algorithm design, efficient data structures, and performance optimization.

We implemented multiple sorting strategies depending on input size:

- Simple Sort (small datasets)
- Medium Sort / Chunk Sort (medium datasets)
- Complex Sort / Radix Sort (large datasets)
- Adaptive Mode (automatic algorithm selection)

The project is built using **doubly linked lists** for efficient stack operations and rotation handling.

Additionally, a **benchmark system** tracks all operations for performance comparison.
